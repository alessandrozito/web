Highlights

In my research, I work towards having easy-to-use software that can be useful to scientists in my field as well as in others. Below is a list of tutorials for packages I developed in parallel with some of my projects.

R-packages

BayesANT - BAYESiAn Nonparametric Taxonomic classifier

BayesANT is a DNA barcoding algorithm to classify DNA sequences within a hierarchical taxonomy, while allowing novel species to be discovered in the test set. See Zito, Rigon, Dunson (2023) MEE.

GitHub Tutorial

BNPvegan - Bayesian nonparametric methods for ecologists

BNPvegan is an R-package that implements the methods described in Zito, Rigon, Ovaskainen and Dunson (2023) JASA

GitHub Tutorial

ConjugateDP - random number generator for the Stirling-gamma distribution

ConjugateDP is an R-package that implements the random sampler for the Stirling-gamma distribution, which is the conjugate prior for the Dirichlet process precision parameter. See methods described in Zito, Rigon, Dunson (2024+) arXiv

GitHub

CompressiveNMF - Bayesian non-negative matrix factorization for mutational signature analysis

This repo contains the code to reproduce the empirical results in Zito and Miller (2024+) arXiv

GitHub
